# GitHub Pitch

## README intro sentence

Composable open-source PLM starter for digital product passports and engineering change impact analysis.

## Resume bullet

Built a FastAPI-based PLM starter that models hybrid BOMs, calculates engineering change blast radius across shared components, and generates digital product passport JSON for lifecycle transparency.

## LinkedIn post draft

I built an open-source PLM starter called ThreadPass PLM. Instead of trying to recreate an entire enterprise PLM suite, it focuses on a narrower but timely problem: turning BOM and ECO data into digital product passports and change-impact insights. The repo includes a FastAPI backend, lightweight dashboard, sample BOM data, CLI, tests, Docker, and CI.
